# Birmingham IMD Domain Maps

LSOA-level maps for overall Index of Multiple Deprivation plus each IMD domain:
- Income
- Employment
- Education
- Health
- Crime
- Barriers to Housing & Services
- Living Environment
- Income affecting children
- Income affecting older people
